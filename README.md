# EDA-Books-Dataset

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a books dataset using Python and Pandas. The analysis helps understand the dataset structure, identify missing values, detect duplicates, and generate statistical summaries.

## Features

- Display first 5 rows of the dataset
- Dataset information analysis
- Statistical summary generation
- Dataset shape analysis
- Column name extraction
- Missing value detection
- Duplicate record detection

## Technologies Used

- Python
- Pandas

## Project Structure

- eda.py
- books.csv
- README.md

## How to Run

1. Install Pandas

bash
pip install pandas


2. Run the script

bash
python eda.py


## Dataset

- books.csv

## Output

The program provides:

- First 5 rows of data
- Dataset information
- Statistical summary
- Dataset dimensions
- Column names
- Missing values report
- Duplicate rows report
